# Product

## Что
Плагин для Claude Code — набор шаблонов и правил, определяющих стандартную структуру C#-проектов.

## Для кого
Для меня (skipper) — автоматизация scaffolding'а новых .NET-проектов с единообразной конфигурацией.

## Зачем
Каждый новый C#-проект должен стартовать с одинаковой базой: analyzers, nullable, тестовый фреймворк, структура папок, agent_docs. Без ручной настройки каждый раз.
